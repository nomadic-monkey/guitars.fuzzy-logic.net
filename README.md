# Fuzzy Logic Guitars

A static e-commerce demo website showcasing guitars for sale.  
Hosted at **https://guitars.fuzzy-logic.net** via Cloudflare Pages.

## 🚀 Deploy Instructions

1. Upload this folder to GitHub as `guitars.fuzzy-logic.net`
2. In Cloudflare Pages, connect to that repo
3. Build command: _none_
4. Output folder: `/` (root)
5. Add custom domain `guitars.fuzzy-logic.net`

That's it — your static site will be live globally with HTTPS!
